# 🧠 Mental Health Support Chatbot

A web-based Mental Health Support Chatbot built using Python and Streamlit that helps users track their emotions, receive motivational support, and improve their mental well-being.

---

## 🌟 Project Overview

The Mental Health Support Chatbot is an interactive application designed to provide emotional support and self-help guidance. Users can share their feelings, and the chatbot responds with supportive messages, coping strategies, and positive affirmations.

This project aims to promote mental wellness by offering tools like mood tracking, journaling, guided meditation, and motivational content in a simple and user-friendly interface.

---

## 🚀 Features

* 💬 Chat-based interaction with users
* 🧠 Emotion detection (Happy, Sad, Stressed, Angry, Bored, Anxious, etc.)
* 📊 Mood tracking system
* 📝 Daily journaling feature
* 🌈 Positive affirmations generator
* 🧘 Guided meditation activities
* 💪 Motivation booster
* 🔊 Text-to-Speech (voice response)
* 🔐 Simple login system
* 📥 Download chat history
* 🆘 Emergency help support

---

## 🛠️ Technologies Used

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/Arpitha-PN/MENTAL_HEALTH_CHATBOT.git
cd MENTAL_HEALTH_CHATBOT
```

---

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

---

### 3️⃣ Run the Application

```
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file and add:

```
OPENAI_API_KEY=your_api_key_here
```
